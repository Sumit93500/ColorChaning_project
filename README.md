# 🎨 Color Changing Project

## 📌 Description

This is a simple web project that changes the background color dynamically using JavaScript.

## 🚀 Features

* Change background color on click
* Random color generation
* Interactive UI

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript

## ▶️ How to Run

1. Download the project
2. Open `index.html` in browser
